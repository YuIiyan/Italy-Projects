# ITworkshop

Welcome to the ITworkshop repository! This project is designed for educational purposes, focusing on teaching Python programming concepts and best practices. Whether you're a beginner or looking to sharpen your skills, this repository provides resources and examples to help you learn effectively.

## Features
- Hands-on Python examples
- Step-by-step tutorials
- Exercises to practice and reinforce learning

## Getting Started
1. Clone the repository:
   ```bash
   git clone git@gitlab.com:marpereira/itworkshop.git
   ```

1. Navigate to the project directory:
   ```bash
   cd itworkshop
   ```
1. Follow the tutorials and explore the examples.
